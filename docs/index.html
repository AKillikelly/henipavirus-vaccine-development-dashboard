Henipavirus vaccine and therapeutics development dashboard

A GitHub Pages-ready dashboard tracking public vaccine, therapeutic, veterinary, and surveillance evidence for Henipaviruses. The active public-health development focus is Nipah and Hendra. Cedar is labelled as a research surrogate/tool, while Ghanaian bat/Kumasi, Mojiang, Angavokely, and Salt Gully rows are surveillance-only unless a source-backed product program appears.

Species snapshots

Highest visible maturity stage by species or scope under the active filters. Surveillance-only rows are retained for traceability but are not shown as active vaccine programs.

Platform tracker

Technology coverage now includes mRNA, self-amplifying mRNA, ChAdOx1 adenovirus, recombinant VSV, recombinant measles vector, subunit protein, VLP, DNA, live-attenuated/replicating-vector, monoclonal antibody, and small-molecule therapeutic approaches.

Clinical trial status and publication endpoints

Trial phase, trial status, completion dates, and publication status are tracked separately. This avoids lumping ongoing trials, completed trials, and published-result trials into the same visual status.

Geographic/site map

Curated trial, manufacturing, preclinical, veterinary, and surveillance locations by country and role. This is a text-based site map suitable for static GitHub Pages deployment.

Maturity board

Cards are placed in the highest public stage supported by curated evidence. Reserve/stockpile readiness, veterinary licensure, trial completion, and publication status are tracked as separate attributes.

Approved veterinary countermeasures

Veterinary products are separated from human products. Equivac® HeV is shown as a licensed horse vaccine in Australia, not a licensed human henipavirus vaccine.

Therapeutics pipeline

Parallel therapeutic development includes monoclonal antibodies and repurposed antivirals. These rows are intentionally separated from vaccine candidates while sharing the same maturity definitions.

Data sources directory

Direct links to registries, CEPI announcements, WHO/R&D Blueprint context, publications, and company/institution pages used by the dashboard.

Stage definitions and visual indicators

Granular stages distinguish preclinical, IND-enabling, Phase I planned, Phase I ongoing, Phase I complete/results pending, Phase I results published, Phase II planned, Phase II ongoing, Phase III, human licensure, and veterinary licensure.

Source audit

The scheduled updater checks configured source pages, structured registries, and publication searches. Broken links, registry status changes, or publication-watch errors become review flags rather than silent curation changes.

Methods and curation policy

This dashboard is a public-evidence curation aid, not a clinical, regulatory, investment, or medical recommendation. Generated by a static GitHub Pages workflow.
